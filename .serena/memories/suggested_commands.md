# Suggested Commands for DA_calc

## Development Commands
- `npm run dev` - Start development server (Next.js)
- `npm run build` - Build production bundle
- `npm run start` - Start production server  
- `npm run lint` - Run ESLint

## NPM Package Management
- `npm install` - Install dependencies
- `npm install <package>` - Add new package
- `npm install -D <package>` - Add dev dependency

## Windows File System Commands
- `dir` - List directory contents (Windows equivalent of ls)
- `type <file>` - Display file contents (Windows equivalent of cat)
- `copy` - Copy files
- `move` - Move/rename files
- `del` - Delete files
- `mkdir` - Create directory
- `rmdir` - Remove directory

## Git Commands
- `git status` - Check current branch and changes
- `git add .` - Stage all changes
- `git commit -m "message"` - Commit changes
- `git push` - Push to remote
- `git pull` - Pull from remote

## TypeScript Commands
- `npx tsc --noEmit` - Type check without emitting files

## File Encoding Note
All files use UTF-8 encoding, important for Japanese text in CSV/YAML files.
